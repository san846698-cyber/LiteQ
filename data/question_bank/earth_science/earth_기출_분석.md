# 지구과학Ⅰ 기출 패턴 분석 (2022~2026학년도)

## 단원별 출제 빈도 (20문항 기준)

| 단원 | 평균 출제 수 | 2점 | 3점 |
|------|-------------|-----|-----|
| 지권의 변동 | 3~4 | 2~3 | 1 |
| 지구의 역사 | 3~4 | 2 | 1~2 |
| 대기와 해양의 변화 | 3~4 | 2~3 | 1 |
| 대기와 해양의 상호작용 | 3~4 | 2 | 1~2 |
| 별과 외계 행성계 | 3~4 | 2 | 1~2 |
| 외부 은하와 우주 | 2~3 | 1~2 | 1 |

## 배점별 특성

### 2점 (12문항)
- 개념 직접 확인: "옳은 것은?", "옳지 않은 것은?"
- 단일 자료 해석: H-R도, 일기도, 지질단면도 하나만 제시
- 보기(ㄱㄴㄷ) 유형 빈출

### 3점 (8문항)
- 복합 자료 해석: 2개 이상 자료 비교
- 계산 포함: 광도비, 반지름비, 후퇴속도, 반감기
- 추론 필요: "이 자료로부터 알 수 있는 것"

## 킬러 패턴 (매회 1~2문항)

| 유형 | 출제 빈도 | 구조 |
|------|-----------|------|
| 별 물리량 복합 | 매회 | H-R도 + L=4πR²σT⁴ + 진화 단계 종합 |
| 지사 해석 복합 | 격회 | 지질단면도 + 부정합 + 상대연령 + 절대연령 |
| 허블 법칙 + 우주론 | 격회 | 적색편이 → 후퇴속도 → 거리 + 우주 성분비 |
| 대기순환 복합 | 간헐 | 기압대 + 해류 + 엘니뇨/라니냐 종합 |

## 보기(ㄱㄴㄷ) 유형
- 전체 20문항 중 약 8~10문항이 보기 유형
- 선지: ①ㄱ ②ㄴ ③ㄱ,ㄷ ④ㄴ,ㄷ ⑤ㄱ,ㄴ,ㄷ (가장 흔한 패턴)
- ㄱ: 기본 확인 (대부분 옳음), ㄴ: 핵심 변별, ㄷ: 함정

## 시각 자료 유형
| 자료 | 해당 단원 |
|------|-----------|
| H-R도 | 별과 외계행성 |
| 지질단면도 | 지구의 역사 |
| 일기도 (지상/상층) | 대기와 해양의 변화 |
| 해수면 온도 분포도 | 대기와 해양의 상호작용 |
| 판 경계 지도 | 지권의 변동 |
| 은하 회전 곡선 | 외부 은하와 우주 |

## 선지 오답 유도 패턴
1. **방향 혼동**: 해류 방향, 판 이동 방향
2. **크기 비교 실수**: 광도 큰 별 ↔ 반지름 큰 별 혼동
3. **인과관계 역전**: 엘니뇨 → 무역풍 약화 (원인↔결과)
4. **시간 순서 오류**: 지사 해석에서 관입/퇴적 순서
5. **공식 오적용**: T⁴ 관계에서 T² 사용
