## 2. 배점별 난이도 차이

### 2점 문항 (1~3번, 매회 3문)
- **사고 단계**: 1단계 (공식 직접 대입)
- **풀이 시간**: 30초 ~ 1분
- **패턴**:
  - `[수학Ⅰ] [2점] [지수 계산]` — 지수법칙으로 값 계산 (8^(2/3) 등)
  - `[수학Ⅱ] [2점] [극한 계산]` — lim 직접 계산 (대입 또는 인수분해)
  - `[확통] [2점] [조합 계산]` — ₙCᵣ 직접 계산

### 3점 문항 (중간 난이도, 매회 14문)
- **사고 단계**: 2~3단계 (조건 해석 → 개념 적용 → 계산)
- **풀이 시간**: 2~4분
- **패턴**:
  - `[수열] [3점] [등차수열의 합]` — a₁, d 조건 주고 Sₙ 구하기
  - `[삼각함수] [3점] [삼각방정식]` — sin(2x + π/3) = 1/2 풀기, 범위 내 해 개수
  - `[미분] [3점] [접선]` — f(x) = x³ + ax에서 기울기 조건 → a 결정
  - `[적분] [3점] [넓이]` — 두 곡선 사이 넓이 구하기
  - `[확률] [3점] [조건부확률]` — P(A|B) 계산, 표/벤다이어그램 활용

### 4점 문항 (고난도, 매회 13문)
- **사고 단계**: 3~5단계 (복합 조건 분석 → 전략 수립 → 다단계 계산 → 검증)
- **풀이 시간**: 4~8분
- **패턴**:
  - `[수열] [4점] [점화식]` — aₙ₊₁ = f(aₙ) 조건식에서 일반항 또는 Sₙ 추론
  - `[미분] [4점] [함수 결정]` — f'(x), f(x) 조건 복합 → 미정계수 다수 결정
  - `[적분] [4점] [정적분 활용]` — ∫₀ᵃ f(t)dt = g(a) 조건에서 f(x) 결정
  - `[확통] [4점] [확률 종합]` — 조건부확률 + 독립시행 결합

### 킬러 문항 (20, 21, 22번 + 29, 30번)
- **특징**: 조건이 3개 이상, 케이스 분류 필요, 함수의 성질 종합 활용
- **대표 유형**:
  - 20번: 수열+조건부 정의 (n의 홀짝에 따른 점화식)
  - 21번: 미분+극값 조건+부등식 (삼차/사차함수 계수 결정)
  - 22번: 적분+함수 결정 (정적분과 미분의 관계 활용)
  - 29번: 확통 킬러 (복잡한 확률 모델)
  - 30번: 확통 킬러 (조건부확률 종합)

## 3. 수식 표현 패턴

### 자주 사용되는 수식 구조
```
- 등차수열: a_n = a_1 + (n-1)d, S_n = \frac{n(a_1 + a_n)}{2}
- 등비수열: a_n = a_1 \cdot r^{n-1}, S_n = \frac{a_1(r^n - 1)}{r - 1}
- 삼각함수: \sin^2\theta + \cos^2\theta = 1, \tan\theta = \frac{\sin\theta}{\cos\theta}
- 미분: f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}
- 적분: \int_a^b f(x)\,dx = F(b) - F(a)
- 극한: \lim_{x \to a} f(x) = L
- 조합: \binom{n}{r} = \frac{n!}{r!(n-r)!}
```

### 수능 문체
- 객관식: "~의 값은?" / "~으로 옳은 것만을 <보기>에서 있는 대로 고른 것은?"
- 단답형: "~의 값을 구하시오."
- 조건 제시: "다음 조건을 만족시키는 ~"

## 4. 선지 설계 패턴

### 객관식 오답 유도 전략
1. **부호 오류**: (-1)ⁿ 처리 실수 → +/- 반대 값
2. **공식 오적용**: Sₙ 대신 aₙ 구함 (또는 반대)
3. **범위 누락**: 이차방정식 해 중 조건 밖 해 포함
4. **계산 실수**: 약분/통분 오류, 자릿수 착각
5. **개념 혼동**: 극값 ≠ 최댓값, 연속 ≠ 미분가능

### 선지 간격 패턴
- 2점: ① 1 ② 2 ③ 3 ④ 4 ⑤ 5 (연속 정수)
- 3점: 값 간격이 일정하지 않음, 계산 실수 유도 값 포함
- 4점: 선지 간격 불규칙, 비슷한 값 2~3개 배치

## 5. 단답형 정답 분포

- **범위**: 0 ~ 999 자연수
- **빈도 높은 범위**: 1~50 (약 60%), 50~200 (약 30%), 200+ (약 10%)
- **자릿수**: 1자리 15%, 2자리 55%, 3자리 30%
- **특이 패턴**: 킬러 단답형은 2~3자리 정수, 계산량이 많아 실수 유도
