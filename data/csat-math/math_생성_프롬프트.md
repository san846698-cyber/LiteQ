## 프롬프트 A: 일간지 (7문항 · 20분)

```
다음 스펙에 맞는 수능 수학 일간지 7문항을 생성해주세요.

### 배점 구성
- 2점 × 2문항 (기본)
- 3점 × 4문항 (표준)
- 4점 × 1문항 (도전)
총 19점

### 유형
- 객관식(5지선다) 4문항 + 단답형(0~999 자연수) 3문항

### 단원 배분
- 수학Ⅰ (지수·로그, 삼각함수, 수열) 3~4문
- 수학Ⅱ (극한·연속, 미분, 적분) 3~4문

### 난이도 가이드
- 2점: 공식 1개 대입, 30초~1분. 교과서 예제 수준.
- 3점: 조건 해석 + 1~2단계 풀이. 2~3분. 수능 3~8번 수준.
- 4점: 복합 조건 + 다단계 풀이. 4~5분. 수능 14~15번 수준.

### 출력 형식 (JSON)
각 문항:
{
  "id": 번호,
  "unit": "수학Ⅰ|수학Ⅱ",
  "topic": "단원명",
  "type": "세부유형",
  "diff": 1~3,
  "pts": 2|3|4,
  "rec": "MM:SS",
  "mode": "choice|short",
  "stem": "한글 발문 (수능 문체)",
  "tex": "LaTeX 수식 (KaTeX 호환)",
  "opts": ["①", "②", "③", "④", "⑤"] or null,
  "ans": 0-based index or "정수 문자열",
  "solution": "한글 풀이",
  "solTex": "LaTeX 풀이",
  "solNote": null or "추가 설명"
}

### 필수 검증
1. 모든 문제를 직접 풀어서 정답 확인
2. 객관식: 정답 유일성 확인
3. 단답형: 0~999 자연수 확인
4. LaTeX: KaTeX 렌더링 가능 확인
5. 오답 선지: 흔한 계산 실수에서 유도된 값일 것
```

```
다음 스펙에 맞는 수능 수학 라이트 모의고사 18문항을 생성해주세요.

### 배점 구성
- 2점 × 2문항
- 3점 × 9문항
- 4점 × 7문항 (이 중 2~3문항 킬러급)
총 56점, 60분 제한

### 유형
- 객관식(5지선다) 13문항 + 단답형 5문항

### 단원 배분
- 수학Ⅰ: 6문항
- 수학Ⅱ: 7문항
- 확률과통계: 5문항

### 킬러 문항 (4점 중 2~3문)
- 수능 20·21·22번 수준
- 조건이 3개 이상, 케이스 분류 필요
- 대표 유형:
  · 조건부 점화식에서 수열 행동 분석
  · f, f', ∫ 조건이 복합된 함수 결정
  · 조건부확률 + 독립시행 종합

(나머지 형식은 프롬프트 A와 동일)
```

## 오답 선지 설계 가이드

### 유형별 오답 패턴
| 단원 | 흔한 실수 | 오답 선지 설계 |
|------|-----------|----------------|
| 지수·로그 | 밑과 지수 혼동 | log₂8 = 2 (정답 3) |
| 삼각함수 | 각도↔라디안 혼동 | sin(30) vs sin(π/6) |
| 수열 | aₙ과 Sₙ 혼동 | Sₙ 구해야 하는데 aₙ 값 넣기 |
| 극한 | 분모→0 처리 실수 | 인수분해 빠뜨리기 |
| 미분 | 도함수 계수 오류 | (x³)' = 2x² (정답 3x²) |
| 적분 | 적분 상수 누락 | 부정적분에서 C 빠뜨리기 |
| 확률 | 여사건 미적용 | P(A) 대신 1-P(A') 계산 누락 |

## 수식 스타일 가이드 (KaTeX)

```latex
% 분수
\dfrac{a}{b}   (display style)
\frac{a}{b}    (text style)

% 적분
\int_a^b f(x)\,dx

% 극한
\lim_{x \to a} f(x)

% 시그마
\sum_{k=1}^{n} a_k

% 삼각함수
\sin\theta, \cos\theta, \tan\theta

% 조합
\binom{n}{r} = {}_{n}\mathrm{C}_{r}

% 절댓값
\left| x \right|

% 루트
\sqrt{x}, \sqrt[3]{x}

% 벡터
\vec{a}, \overrightarrow{AB}
```
